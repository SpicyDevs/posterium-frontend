---
category: Integrations
question: What is "Copy for AIOMetadata"?
order: 2
---

AIOMetadata is a popular Plex metadata agent that replaces default artwork with custom sources. The "Copy for AIOMetadata" button in Posterium copies a URL template pre-formatted with `{imdb_id}` placeholders.

That way AIOMetadata injects each title's IMDb ID automatically — one template, every movie gets its own poster. No per-title copy-pasting.
